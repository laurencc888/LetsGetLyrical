# Let's Get Lyrical!

Completed as a semester-long project for USC's CSCI310, Software Engineering. Development was partitioned into four sprints, during which we would develop the website, meet with our stakeholder, and document our progress. As the team's project manager, I would lead meetings with (internal and with our stakeholder), build detailed schedules for our team's success, take on the brunt of tasks, and check-in with each of our team members.

## This repository features:
- source code
- end-to-end tests (Jest, JUnit, Gherkin)
- design mockups
